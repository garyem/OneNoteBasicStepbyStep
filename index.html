Desktop App Setup

1
Open OneNote Desktop App
Start Menu ‚Üí Type "OneNote" ‚Üí Click "OneNote" (not OneNote for Windows 10)
OneNote opens with a purple interface showing recent notebooks or a welcome screen
2
Create Your First Notebook
File ‚Üí New ‚Üí Choose location (OneDrive recommended) ‚Üí Type notebook name "Work Notes"
A dialog box appears asking for notebook name and storage location
3
Create the Notebook
Click "Create Notebook" button (blue button at bottom right of dialog)
OneNote creates the notebook and opens it. You'll see "Work Notes" at the top, with one default section called "New Section 1"
4
Rename the Section
Right-click on "New Section 1" tab ‚Üí Select "Rename" ‚Üí Type "Meetings"
The section tab now shows "Meetings" instead of "New Section 1"
5
Create Your First Page
Press Ctrl + N OR click "Add Page" on the right side
A new blank page appears with a cursor in the title area at the top
6
Name Your Page
Type "Meeting Template" in the title area ‚Üí Click anywhere below to start adding content
The page title shows "Meeting Template" and cursor moves to the content area

Web Version Setup

1
Open OneNote Web
Go to onenote.com ‚Üí Click "Sign in" ‚Üí Enter your Microsoft account credentials
You'll see the OneNote home page with any existing notebooks or a "Create notebook" option
2
Create New Notebook
Click "+ New notebook" (purple button) ‚Üí Type "Work Notes" in the name field
A dialog appears with a text field for the notebook name
3
Create the Notebook
Click "Create" button (purple button at bottom of dialog)
OneNote opens your new notebook with a default section and blank page ready
4
Rename the Section
Right-click on the section tab ‚Üí Select "Rename Section" ‚Üí Type "Meetings"
The section tab updates to show "Meetings"
5
Name Your First Page
Click in the title area (says "Untitled page") ‚Üí Type "Meeting Template"
The page title changes to "Meeting Template" and appears in the page list on the right

1
Prepare Your OneNote Page

2

üìãSection 3: Working with Meeting Transcripts

1
Download Transcript from Teams
Teams ‚Üí Calendar ‚Üí Your meeting ‚Üí Recording and transcript ‚Üí Click "Download transcript"
A .vtt or .docx file downloads to your computer containing the full transcript with timestamps and speaker names
2
Copy Transcript to OneNote
Open the downloaded transcript file ‚Üí Select all text (Ctrl + A) ‚Üí Copy (Ctrl + C) ‚Üí Go to OneNote ‚Üí Paste (Ctrl + V)
The full transcript appears in OneNote with speaker names and timestamps preserved
3
Find Action Items in Transcript
Press Ctrl + F ‚Üí Search for keywords like "will do", "action", "need to", "follow up", "assign"
OneNote highlights all instances of your search term throughout the transcript
4
Extract Key Quotes
Highlight important text in transcript ‚Üí Copy (Ctrl + C) ‚Üí Click elsewhere on page ‚Üí Paste ‚Üí Add context or action needed
You now have key quotes separate from the full transcript for easy reference
5
Create Action Items with Checkboxes
Type the action item ‚Üí Press Ctrl + 1 ‚Üí Add person's name with @
A checkbox appears next to your action item, and you can check it off when completed
6
Tag Important Items
Select text ‚Üí Press Ctrl + 2 for Important tag OR Ctrl + 3 for Question tag
Colored tag icons appear next to your text, making them easy to find later

üìÅSection 4: Organizing Your Meeting Notes

1
Create New Sections for Different Teams/Projects
Right-click on existing section tab ‚Üí "New Section" ‚Üí Type section name (e.g., "Team Alpha", "Project Beta")
A new section tab appears next to your existing ones
2
Move Pages Between Sections
Right-click on page title in page list ‚Üí "Move or Copy" ‚Üí Select destination section ‚Üí Click "Move"
The page disappears from current section and appears in the destination section
3
Create Subpages for Follow-ups
Right-click on a page ‚Üí "New Subpage" ‚Üí Name it "Follow-up Actions" or "Next Steps"
A subpage appears indented under the main page in the page list
4
Use Search to Find Old Notes
Press Ctrl + E ‚Üí Type keywords from the meeting you're looking for
Search results show all pages containing your keywords, with preview text highlighting the matches
